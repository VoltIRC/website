How to Embed a Chat Client

The following instructions will walk you through embedding a chat client in your website. Installing a chat client is a great idea for online communities who wish to enable their users to chat in real time.

To get started, visit our chat client with a desktop web browser: http://webchat.voltirc.com:9090/

Once you have opened the chat client, click the blue menu button in the top left corner.

Select "Add webchat to your site" to start the web chat wizard.

You will be prompted to enter a nickname to use. If you wish for users to select their own nickname, leave this field blank. If you would like your users to use a randomly generated nickname that is specific to your communitiy, use something like "MyCommunity.....". Users who do not fill out their own nickname will now join your chat room with a nickname prefixed with "MyCommunity" followed by five random numbers. Ex. MyCommunity52847

Next, select the channel you would like your users to join. Please note that all channel names must start with # such as #chat or #help. You can select multiple channels by separating the channel names with a comma. Ex. #mychat,#yourchat,#ourchat

Next you will be asked if you want to display a "Connect Dialog". If you want your users to have an opportunity to change their nickname before joining your chat room, select "Show the connect dialog." If you'd rather your users connect directly without giving them a chance to edit their nickname, select "Connect without displaying the dialog."

You will then be asked if you wish to customize the colors used in the web chat. This is a great feature that enables you to match the web chat widget to your website's color scheme. If you're not sure which colors to use, select the default option. Otherwise, select the colors you wish to use for the webchat. Remember, you can always change these colors later.

The web chat client will generate a code for you to embed. Copy and paste this code into your website. You're all set! Users who visit your website will now be able to use a chat client to connect with other users!

If at any time you require assistance with the chat client or getting it set up, please visit us in #help.
